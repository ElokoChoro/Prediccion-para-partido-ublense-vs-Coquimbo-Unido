# Predicción para el partido Ñublense vs Coquimbo Unido (19/04/2025)

En el archivo `ñublense_vs_coquimbo_unido.py` armé un modelo de Regresión Lineal usando `scikit-learn` de Python. Básicamente, agarré datos recientes y reales sobre cómo vienen jugando los equipos (sus rachas de victorias, el promedio de goles en el campeonato y si son locales o visita) y armé un set de datos para entrenar al modelo.

Mi idea aquí fue aterrizar un poco la teoría y hacer algo práctico: procesar la data en matrices y sacarle al modelo una probabilidad de quién va a ganar el partido de este 19 de abril.
